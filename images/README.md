Images
=======

Contains the images that the chaser application will use.  It includes
the original chaser (i.e. neko) but others can be downloaded.  To ease
use, all images are converted from their original formats to PNG (with
masks.)

# Perquisites
- OSX 10.11+MacPorts
     - packages
          - ImageMagick
          - py-pil
          - icoutils
          - unzip
- Ubuntu 15.04, Raspbian GNU/Linux 8
     - packages
          - ImageMagick
          - python-pil
          - icoutils
          - unzip

# Download
- `make` - download and convert all images sets
