Arduino + Adafruit SSD1306
==========================

# Perquisites

## Hardware
- SSD1306
- Arduino capable of I2C and/or SPI
    - tested ArduinoUNO+SSD1306/I2C

## Software
- Arduino IDE >=1.6
- IDE Libraries
    - Adafruit-GFX-Library
    - Adafruit_SSD1306

# Build
- `make` to create chaser images.
- Arduino IDE to compile+install sketch.
