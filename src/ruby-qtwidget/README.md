Ruby+FFI+QT::Widget
===================

Perquisites
-----------
- UBUNTU 15.04
    - GEMs
        - ffi
    - packages
        - ruby-qt4

Build
-----
- `make` - copy images and build chaser shared object

Execute
-------
- `ruby -I . main.rb`
